# 서브쿼리(Subquery)
* 하나의 쿼리 안에 존재하는 또 다른 쿼리
* 밖에 있는 쿼리를 메인 쿼리라 한다.
* 위치에 따라 나뉘는 서브쿼리
  * SELECT 절 - 스칼라 서브쿼리
  * FROM 절 - 인라인 뷰
  * WHERE 절, HAVING 절 - 중첩 서브쿼리(nested subquery)
* ORDER BY 절,INSERT 문의 VALUE 절 
* 다중 행 서브쿼리의 경우 `=` 조건과 함께 사용할 수 없다.
* 다중 컬럼 서브쿼리의 경우 IN 절과 함께 사용할 수 있다.
* 메인 쿼리의 컬럼이 포함되지 않은 서브쿼리를 비연관 서브쿼리라고 한다.
