# 트랜잭션(Transaction)
 
* 개념
  * 데이터를 조작하기 위한 하나의 논리적인 작업 단위
  * 작업 수행에 필요한 SQL 문들의 모임으로 이해해도 좋음


# NULL
* 존재하지 않음
* 값이 없음
* 숫자 0과는 의미 자체가 다르다.
* 연산
  * 가로 연산
    * NULL이 포함되어 있으면 결과값은 NULL이 된다.
    * EX.SELECT REVENUE,OUTMONEY FROM TABLE
  * 세로 연산
    * 다른 인스턴스의 데이터와 연산할 때 NULL 값을 제외한다.


    * ex. SELECT SUM(REVENUE) FROM TABLE
